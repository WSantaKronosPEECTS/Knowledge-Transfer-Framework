## Schnellstart

Willkommen beim Wissenstransfer-Framework v1.0. Dieses Framework bietet eine umfassende Lösung für das Wissenstransfer-Lernen und kombiniert drei leistungsstarke Module:

- **KTLS-MT**: Wissenstransfer-Lernsystem - Multi-Task
- **CEUT**: Umgebungsübergreifender Einheitlicher Transfer
- **ETT-2**: Erweiterte Transfertechniken - Version 2
