## Inicio Rápido

Bienvenido al Framework de Transferencia de Conocimiento v1.0. Este framework proporciona una solución integral para el aprendizaje por transferencia de conocimiento, combinando tres módulos potentes:

- **KTLS-MT**: Sistema de Aprendizaje por Transferencia de Conocimiento - Multi-Tarea
- **CEUT**: Transferencia Unificada de Entornos Cruzados
- **ETT-2**: Técnicas de Transferencia Mejoradas - Versión 2
