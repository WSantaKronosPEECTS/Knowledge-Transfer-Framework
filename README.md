# Knowledge Transfer Framework

Knowledge Transfer Framework (KTLS-MT + CEUT + ETT-2) v1.0

## Overview

A comprehensive framework for knowledge transfer learning, combining three powerful modules:

- **KTLS-MT**: Knowledge Transfer Learning System - Multi-Task
- **CEUT**: Cross-Environment Unified Transfer
- **ETT-2**: Enhanced Transfer Techniques - Version 2

## Project Structure
