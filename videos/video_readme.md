# Educational Video Package - Knowledge Transfer Framework

## Overview
This directory contains all assets and materials for the Knowledge Transfer Framework educational video package (v1.0). The video provides a comprehensive introduction to the framework's key concepts including Elastic Time Theory, Consciousness-Enhanced Understanding Tools (CEUT), and multi-level educational integration.

## Package Contents

### Main Files
- **final_video_script.md** - Complete narration script with scene breakdowns and production notes
- **subtitles_eng_es.srt** - Bilingual subtitles in English and Spanish (SRT format)
- **animations.json** - Technical specifications for all animations used in the video
- **video_readme.md** - This file

### Assets Directory
- **storyboard.md** - Visual storyboard and scene composition details
- **background-music-license.md** - Licensing information for background music
- **scene-stills/** - Directory containing still frames from each major scene

## Usage Instructions

### For Video Editors
1. Use `final_video_script.md` as the narration guide
2. Reference `storyboard.md` for visual composition
3. Import `animations.json` into your animation software
4. Add subtitles using `subtitles_eng_es.srt`
5. Ensure compliance with music licensing terms in `background-music-license.md`

### For Translators
The subtitle file follows standard SRT format with bilingual entries. To add additional languages:
1. Copy the English lines
2. Replace with translated text
3. Maintain timing codes
4. Save with appropriate language code (e.g., subtitles_eng_fr.srt)

### For Content Reviewers
- Scene stills provide quick visual reference for content verification
- Script includes timestamp markers for precise scene location
- All materials are version controlled for tracking changes

## Technical Specifications
- Video Format: 1920x1080 (Full HD)
- Frame Rate: 30 fps
- Duration: ~10 minutes
- Audio: Stereo, 48kHz
- Subtitle Format: SRT (SubRip)
